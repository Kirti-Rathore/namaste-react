Ans: Library : - A Library is a collection of pre-written code or functions that can be reused in different projects. - It provides specific functionality or features that you can call upon in your code when needed. - You have more control over your code and you can choose which parts of the library to use. - libraries are like tools you can use to perform specific tasks within your code. Framework : - A framework is a pre-established structure or set of rules that dictate overall architecture and flow of your application. - Developers need to follow the framework's design principles. - Frameworks are like blueprints that define the overall structure and flow of your application. 3.
